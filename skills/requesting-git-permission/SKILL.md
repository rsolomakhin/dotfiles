# Skill for requesting permissions for git operations

There is no need to pause and ask the user for read-only commands.

## Instructions

1.  If there is a need to run the following read-only commands, then it is OK to
    execute these commands without requesting user permissions:
    -   `git status`
    -   `git log`
    -   `git diff`
